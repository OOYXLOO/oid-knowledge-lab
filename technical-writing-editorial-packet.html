Publication boundary

This repository does not claim a full OID-base mirror, a live Backblaze B2 upload, or a live Genblaze generation run unless a specific public artifact says so. The editorial material is built from public sample data, static proof pages, and locally verifiable reports.

Readable

Samples are designed for editor review first: concrete audience, narrow promise, clear source boundary.

Verifiable

Public pages point back to local commands, generated reports, and source snapshots where possible.

Reusable

The same evidence-log pattern can support documentation, AI review, generated media, and client handoff.
