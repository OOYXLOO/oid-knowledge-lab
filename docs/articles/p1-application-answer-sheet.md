## Safe submission boundary

- No credentials.
- No private account exports.
- No private customer inventories.
- No payment, tax, or identity data.
- No copied full third-party page bodies.
- No claim that any sample has already been accepted by an editor.
- Any accepted assignment should be written freshly for the accepted brief, current style guide, originality policy, and editor feedback.
