## Short Role Summary

```text
I review AI-generated code as an untrusted draft. I define expected behavior,
write focused tests, check edge cases and unsafe inputs, compare alternative
model responses, and explain whether a solution should be accepted, revised, or
rejected.
```

## Review Method

1. Restate expected behavior.
2. Identify the main failure mode.
3. Check normal, boundary, invalid, and unsafe inputs.
4. Revise narrowly when the generated answer is close but unsafe.
5. Explain the accept / revise / reject decision with evidence.
