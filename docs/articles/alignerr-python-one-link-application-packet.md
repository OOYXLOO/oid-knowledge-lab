## Work Method

1. Translate ambiguous AI output into expected behavior.
2. Identify the validation surface: code, data, prompt result, or pipeline artifact.
3. Add focused checks for normal, boundary, invalid, and unsafe cases.
4. Keep source facts, generated claims, reviewer findings, and residual risk separate.
5. Produce a short decision: accept, revise, or reject.
