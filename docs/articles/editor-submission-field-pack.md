## Short bio

I write implementation-backed developer tutorials about data workflows, observability handoffs, release-safety checks, and public proof surfaces. My samples are backed by runnable Node.js scripts, generated reports, GitHub Pages artifacts, dataset manifests, and clear boundaries around credentials, private inventories, and copied third-party page bodies.
