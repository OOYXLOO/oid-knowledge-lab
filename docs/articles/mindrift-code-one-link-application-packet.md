# Mindrift Code One-Link Application Packet

This packet is a compact first link for AI code-review applications, especially roles that ask reviewers to evaluate, improve, and explain AI-generated code.

## Short Role Summary

I review AI-generated code as an untrusted draft. I define expected behavior, run focused tests, check edge cases and unsafe inputs, compare alternatives, revise unsafe implementations, and write concise evidence-backed verdicts.

## Best Fit

- Python code review
- Generated-code correctness evaluation
- Model response comparison
- Bug finding and edge-case review
- Corrected implementation writing
- Test-case design
- Evidence-backed written feedback

## Review Method

1. Restate the expected behavior.
2. Identify the main failure mode.
3. Check normal, boundary, invalid, and unsafe inputs.
4. Revise the implementation narrowly.
5. Add focused tests.
6. State a verdict and residual risk.
