# OpenTrain AI Code Evaluation One-Link Packet

This packet is a compact first link for AI trainer and code-evaluation applications.

## Role Summary

I evaluate AI-generated code as an untrusted draft. I define expected behavior, write focused tests, check edge cases and unsafe inputs, compare model responses, and explain whether a solution should be accepted, revised, or rejected with evidence.

## Best-Fit Tasks

1. Python code evaluation.
2. Model-response comparison.
3. Training-data review.
4. Bug and edge-case finding.
5. Corrected implementation writing.
