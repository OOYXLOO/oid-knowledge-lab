## Copy-ready application answer

Short role summary:

I review AI-generated code as an untrusted draft. I define expected behavior, run focused tests, check edge cases and unsafe inputs, compare alternatives, revise unsafe implementations, and write concise evidence-backed verdicts.

Review method:

1. Restate the expected behavior before judging the generated answer.
2. Identify the main failure mode.
3. Check normal, boundary, invalid, and unsafe inputs.
4. Compare candidate responses against behavior and evidence.
5. Revise the implementation narrowly when a fix is needed.
6. Add focused tests and state residual risk.
