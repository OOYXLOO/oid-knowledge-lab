# Contributor Opportunity Index

This index collects neutral, public-facing writing samples and tutorial packets for editors who want implementation-backed developer content.

## Core Packets

- Envato Tuts+ Code Contributor Packet: two browser-only JavaScript tutorial ideas with working proof links.
- Unleash Topic Fit Brief: feature-flag tutorial framing with a continuous-delivery angle.
- Appsmith Template Blueprint: a compact internal-tool pattern that can become a tutorial or integration example.
- Earthly Build Workflow Proof: reproducible build and artifact review workflow sample.
- KnowledgeOwl Editorial Fit Brief: evidence-log workflow for reviewing AI-assisted help content.

## Sample Themes

- Browser-only review dashboards.
- Public-safe evidence handoffs.
- Source-boundary checks for AI-assisted drafts.
- Build, release, and documentation review workflows.
- Lightweight JavaScript tools that do not require private data or account credentials.

## Boundaries

These materials are written as public portfolio assets. They avoid credentials, private customer data, account screenshots, payment information, personal identifiers, unpublished client code, and copied third-party article bodies.
