# Data Directory

`data/sample/` contains a small verification sample generated by the crawler. Parsed sample JSON/JSONL, crawl state, reports, and raw Markdown debug files are local-only and ignored; only `RUN-*.md` receipts are intended for Git.

`data/full/` is ignored by Git and should only be used after receiving a clear authorization basis for full collection.

Publishable aggregate outputs and sitemap-level catalogs live under `reports/`; generated raw JSONL and raw page files stay out of Git unless the source license explicitly allows publication.
