# Demo Video Page Design

## Visual Identity

Media Ledger Studio's demo walkthrough uses the same operational dashboard language as the app:

- Background: cool off-white `#f7f8f6`
- Text: deep slate `#172326`
- Dark panel: green-black `#102326`
- Accent: coral `#e86f51`
- Secondary accent: teal `#2a8f8a`

## Typography

The walkthrough uses system UI for practical product fidelity and monospace only for object keys, model names, and checksums. It avoids decorative title-card typography because the goal is a clear product review, not a cinematic trailer.

## Motion

The page uses deterministic CSS keyframe timing. Each scene fades and slides in sequence, matching the written 3-minute script in compressed reviewer-friendly form. It is intended as a browser-playable walkthrough and as a source page for later screen recording.

## What Not To Do

- Do not add revenue or prize language.
- Do not show private media or credentials.
- Do not use fake customer names as real references.
- Do not use heavy gradients or decorative glow as the primary design.
