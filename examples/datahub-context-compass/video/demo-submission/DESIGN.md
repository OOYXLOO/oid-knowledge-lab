# DataHub Context Compass Demo Design

Concept angle: a metadata control room where every proposed automation has to
pass through lineage, owner, PII, freshness, and incident checks before it is
allowed to act.

Visual identity:

- Background: deep ink `#091014`
- Foreground: paper mint `#e7fff4`
- Accent A: catalog green `#22c55e`
- Accent B: warning amber `#f59e0b`
- Danger: red clay `#ef4444`
- Type: League Gothic for display, IBM Plex Mono for data labels and body

Density plan:

- Every scene has a persistent grid, scanline texture, registry labels, and
  edge metadata.
- Main content alternates between large caption panels, dataset cards, risk
  meters, MCP call rows, and submission proof.
- Transitions use fast data-sweep wipes with simultaneous outgoing/incoming
  motion.
