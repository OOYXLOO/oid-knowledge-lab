What This Shows

Slack-style review messages become separated source facts, draft claims, reviewer checks, blockers, and privacy boundaries.
The generated Markdown handoff makes publication status visible before an answer or article is shipped.
Inputs stay local to the browser in the playground; no workspace installation is needed for the public demo.
The core parser is also exposed as an MCP-style tool for future assistant and workflow integrations.
