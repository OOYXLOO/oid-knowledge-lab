Instructor pitch

Build and Ship a Safe Static Evidence Dashboard in Node.js

The course teaches a complete project: generate JSON and Markdown evidence reports, render a static dashboard, write a manifest, and add a publish guard before sharing public artifacts. The same outline can be shortened into a written tutorial or expanded into a 30 to 40 minute screencast course.

Course outline

Seven lesson path

Lesson 1Preview the finished dashboard

Show the final public page, generated reports, source repository, and safe-publication boundary.

Lesson 2Set up the report schema

Create the small data shape that will drive JSON, Markdown, and HTML outputs.

Lesson 3Generate artifacts

Write Node.js scripts that turn sample input into structured output and a reviewable handoff.

Lesson 4Render the static site

Build the dashboard page, proof links, and reviewer routing surface.

Lesson 5Add tests and checks

Use syntax checks, assertions, and dataset manifests to keep the output reproducible.

Lesson 6Add a publish guard

Block credentials, local-only crawls, copied page bodies, and unsafe generated artifacts.

Lesson 7Ship the review packet

Build the site, verify links, and package the final proof set for an editor or reviewer.
