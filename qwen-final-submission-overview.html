A source-grounded Qwen Cloud candidate for turning sanitized Object Identifier evidence into deterministic checks, model-assisted remediation notes, and human-approved handoff artifacts.

Demo reel Repository

What it does

The project starts from a safe OID inventory, classifies values with deterministic code, maps public registry evidence, and prepares a remediation queue that Qwen can explain in reviewer-friendly language.

How Qwen is used

Qwen is the language and reasoning layer after source checks. It drafts summaries and next actions from a sanitized packet; it does not decide source truth or take external action.
