Idea 1: Local-First Provenance Ledger

AI-generated images, diagrams, audio, and video assets are easy to create but hard to hand off responsibly. This tutorial shows how to build a browser-only JavaScript ledger with asset records, review status, model notes, search filters, live summary metrics, and JSON or Markdown export.

  • Design a minimal provenance data model.
  • Build a dependency-free HTML and JavaScript review app.
  • Export a delivery sheet for editors, reviewers, or clients.

Idea 2: Browser-Only Evidence Dashboard

Many developer workflows depend on public registries, but teams often copy identifiers into docs, agents, or dashboards without keeping source links and review notes. This tutorial shows how to build a searchable static evidence dashboard from structured public records.

  • Convert registry-like records into a searchable static dataset.
  • Link records to public sources and reviewer notes.
  • Keep private data and credentials out of public demos.
