What is measured

Baseline scan versus indexed evidence lookup

The benchmark loads a small public-safe review-event fixture, runs repeated evidence questions, and compares a simple scan path with an indexed path that caches normalized tokens. It records runtime, average iteration time, heap delta, platform, architecture, Node.js version, and a clear claim boundary.

The included GitHub Actions workflow verifies the benchmark as a baseline by default. A real Arm submission must rerun the same harness on an Arm64 environment and pass the stricter --require-arm64 check.

Workflow badge and the latest public baseline run are linked above for auditability.
