Live-run checklist

What remains before real provider execution

The public repository stays credential-free. A live adapter run would require the following environment variables to be present in a private execution environment, not committed to source control.

VariablePurpose
B2_APP_IDBackblaze B2 application key id
B2_APP_VALUEBackblaze B2 application key value
B2_BUCKET_NAMETarget B2 bucket for media and sidecar objects
GENBLAZE_AUTH_VALUEGenblaze authorization value
GENBLAZE_ENDPOINTGenblaze generation endpoint
