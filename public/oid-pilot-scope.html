Pilot shape

What the first engagement includes

AreaConcrete boundaryAcceptance check
Input20 to 100 sanitized CSV or TSV rows with an oid column and safe labels.No credentials, customer names, private hostnames, raw account exports, or copied page bodies.
ClassificationEach row is marked invalid, evidence-ready, known enterprise, unknown enterprise, or unresolved.Every row has a status and a next action.
EvidencePublic IANA PEN data and OID-base sitemap-level metadata are used as source references.The handoff links source evidence without copying OID-base page bodies.
DeliveryMarkdown summary, remediation CSV, public-source evidence map, and re-run notes.The owner can import the queue, assign actions, and re-run the checks after cleanup.

Copyable reply

Short first-response text

Use this when a technical owner has an OID spreadsheet, certificate policy list, MIB inventory, or registry cleanup problem.

Hello, I can start with a small sanitized OID inventory pilot.

Please share 20-100 rows as CSV/TSV with an oid column and non-sensitive labels only.
I will classify each row, separate malformed values and unresolved owner questions, link public registry evidence where available, and return a Markdown summary plus a remediation CSV.

The first pass does not need credentials, private account exports, customer names, production hostnames, or raw logs.
Pilot scope: https://ooyxloo.github.io/oid-knowledge-lab/oid-pilot-scope.html
Sample handoff: https://ooyxloo.github.io/oid-knowledge-lab/sample-assessment.html

When to expand

Decision rules after the pilot

ResultDecisionNext step
Most rows are evidence-readyKeep the scope small.Deliver the final summary and re-run notes; no larger export is needed.
Many rows need owner reviewExpand only with approval.Agree on a private delivery location and a larger sanitized export.
Inputs contain private dataStop and sanitize first.Remove secrets, customer identifiers, account exports, hostnames, and raw operational logs.
The request becomes live remediationCreate a new scope.Separate assessment work from production changes, credentials, or emergency operations.
