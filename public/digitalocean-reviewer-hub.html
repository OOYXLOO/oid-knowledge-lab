Tutorial idea

Build a Static Evidence Dashboard with Node.js and a Publish Guard

The proposed tutorial would show developers how to create a small Node.js reporting workflow, generate JSON and Markdown artifacts, build a static HTML dashboard, add a dataset manifest, and block unsafe generated files before publishing. It is designed as a practical, reproducible workflow rather than a product announcement.
