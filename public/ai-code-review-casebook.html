Review case

duration_parser_review

This public case starts from a generated duration parser that silently returns 0 for unknown units. The review rejects that behavior, revises the parser to raise explicit errors, and proves the safer behavior with focused unit tests.

Boundary

Public and account-safe

The examples are public, original review artifacts. They do not include private source code, credentials, account exports, identity documents, payment data, tax records, private platform messages, or restricted assessment content.
