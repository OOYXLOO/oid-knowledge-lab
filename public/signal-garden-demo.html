Interactive Reddit game prototype

Signal Garden turns a Reddit post into a tiny daily puzzle ritual.

Rotate signal tiles, connect the root to the bloom, copy a route code into the thread, replay another player's path, compare routes, and return for a new seeded board. This page collects the current public demo video, screenshots, and source branch for reviewer handoff.
